# Agents and Responsibilities

## Planner (Current Phase)
- Create structured plans and documentation in docs/.
- Do not modify source code in planning phase.
- Identify risks, dependencies, and validation strategy.

## Build Model (Implementation Phase)
- Execute refactors and test runs.
- Keep changes small and reversible.
- Run tests after each refactor slice.
- Update docs/TODO.md status per task.
- Produce phase deliverables and request confirmation before moving on.

## Test Runner
- Run unit, integration, and coverage tests as defined.
- Report failures with minimal reproduction steps.

## Documentation Maintainer
- Maintain phase reports in docs/.
- Update README only after all phases complete.
